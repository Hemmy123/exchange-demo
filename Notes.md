# Order Book Notes


### Requirements:
An order book needs to be able to hand the following situations
- bid(instrumentID, price, amount)
- ask(instrumentId, price, amount)
- print // print entire order book
- modify(orderType, instrumentId, price, amount)
- cancel(orderType, instrumentId, price, amount)
- 


### AI Assisted areas
- CMake files


### Glossary
*Order* - A single instruction for the order book, this could be a bid, or ask
*Market Order* - buy/sell at whatever the market rate is.
*Limit Order* - Buy/sell at a specific value. the buy/sell only happens if the instrument reaches that value.
